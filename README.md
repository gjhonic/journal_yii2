**Instruction manual**

1) Loading
```
git clone https://github.com/gjhonic/journal_yii2.git
```

2)To download the packages
```
cd app
composer update
```
3)Build docker
```
cd ..
docker-compose up
```

4) localhost:8080 - app
5) localhost:1234 - phpMyAdmin
6) bd - dump database
